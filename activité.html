Le language HTML

HTML est l'abréviation de Hypertext Markeup Language, soit en français : langage hypertexte de balises

Pour que la page soit valide l'élement Head doit contenir un élèment Title et le meta charset qui prendra en valeur le type d'encodage choisi pour les langues latins , nous choisirons le type UTF-8
